## サービス概要
声が出せず、身体も動かせず、瞬きだけで意思表示する方のための、「質問テンプレート×ログ共有」ができる意思疎通支援アプリです。
家族・介護スタッフが本人の気持ちや体調を理解し、安心してケアできる環境をつくる手助けをします。

## このサービスへの思い・作りたい理由
- 妻の義父が脳卒中で倒れ、介護施設で寝たきりの状態となりました。
- 言葉を発することはほとんどできず、意思を伝える手段は「目を閉じる・開く」といった瞬きだけです。
- 「暑い」「寒い」「痛い」といった体調変化がすぐに分からず、表情だけでは判断できないもどかしさがありました。
- この状況を少しでも改善するため、本人が最小限の負担で意思を示せて、家族や介護者が迷わず理解できるコミュニケーション手段を作りたいと考え、このアプリの制作を決めました。
- このサービスを通して、家族の負担を減らすだけでなく、本人の「伝えたい気持ち」や「生活の選択肢」を少しでも取り戻すきっかけになればと思っています。

## ユーザー層について
本サービスは、以下のユーザー層を対象としています。

- 発声が困難で意思表示が瞬きに限定される方とそのご家族

脳卒中・ALS・神経疾患などにより言葉を発することが難しく、手足も動かしづらい方は、意思表示の手段が非常に限られています。
画面を見ることができ、瞬きで応答が可能であれば利用できます。

- 介護施設のスタッフ

介護現場では、一人の利用者の体調・希望・生活のこだわりを正確に把握することが難しいケースがあります。
回答ログが共有できることで、スタッフ間での状態共有がしやすくなり、ケアの質が向上します。

## サービスの利用イメージ
1. 家族や介護スタッフがアプリを起動し、カテゴリ一覧を要介護者に見せます
2. 要介護者が瞬きで希望するカテゴリを伝え、家族がそれをタップします
3. 選択されたカテゴリ内の質問が表示され、要介護者が瞬きで回答します
4. 家族が回答をタップすることで記録されます
5. 回答はログとして保存され、家族や介護職員がいつでも確認できます

## ユーザーの獲得について
- 家族・介護者からの利用開始
家族や介護スタッフが利用することで、施設内の他の職員にも広まり、利用者の日々のログを共有するツールとして活用してもらうことを想定しています。

- SNSでの紹介
利用方法や開発背景をブログやSNSで発信することで、同じ悩みを抱えている人の目に触れやすくなります。
特に「瞬きだけで使える」「家族の負担が減る」という特徴は共感されやすいと考えています。

## サービスの差別化ポイント・推しポイント
既存のサービスで、カテゴリ別の質問テンプレートがあり、質問を選択することでコミュニケーションをとれるアプリがあります。
しかし、カテゴリーが限定的で限られ、利用者がカスタマイズできないため使いづらさがあります。
(例えば、既存アプリには「トイレに行きたい」という選択肢が固定で用意されていますが、下半身の感覚がない方にとってはその項目が不要です。逆に、「暑い/寒い」といった体調に関する質問を増やしたいというニーズもあります)

このアプリでは利用者が質問をカスタマイズできるため、利用者の状況にあった質問がスムーズにできます。
また、質問の回答をログとして記録できるため、過去の要望を可視化することができ、周囲と状態の共有も可能です。

- カスタム質問の作成機能
- ユーザー認証機能

- 自動音声読み上げ機能(Web Speech APIを使用予定)
- 質問の並び替え・お気に入り登録

- 写真・動画アップロード機能
- カレンダー・メモ機能
- 家族アカウント管理
- 複数人の家族がログを共有できるようユーザー管理を実装
- あいうえお表の実装
- AIによる回答ログの分析・可視化(実装方法は検討中)
